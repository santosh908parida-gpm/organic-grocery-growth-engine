
# Organic Grocery Growth Engine

## Overview

This project is a Growth Product Management case study using grocery order data to analyze organic product adoption.

The goal is to understand how users move from trying organic products to becoming high-value organic buyers, and to design growth experiments that increase organic basket share, repeat purchase, and category expansion.

## Business Problem

Many users buy organic products, but not all become high-value organic buyers.  
The project identifies where users drop off and which product/category strategies can improve high-value conversion.

## Dataset

The project uses public online grocery order data containing:

- Orders
- Products
- Aisles
- Departments
- Reorder behavior
- User-level purchase history

## Methodology

1. Data loading and cleaning
2. Product, aisle, and department merging
3. Organic product tagging
4. User-level behavior engineering
5. AARRR funnel analysis
6. User segmentation
7. Product/category opportunity analysis
8. Growth experiment roadmap using ICE scoring

## Key Metrics

- Organic item share
- Organic order share
- Organic order frequency
- Organic reorder rate
- Funnel conversion rate
- Drop-off rate
- High-value organic user conversion

## Key Insight

The biggest growth opportunity is not first-time organic trial.  
The largest leakage occurs before users become high-value organic buyers.

The strategy should focus on converting loyal organic buyers into high-value organic buyers through basket expansion, category cross-sell, replenishment nudges, and personalized organic alternatives.

## Growth Experiments

- Organic Basket Expansion Recommendations
- Complete Your Organic Basket Bundle
- Organic Replenishment Reminder
- Organic Alternative Swap Campaign
- High-Value Organic Upgrade Offer
- Organic Membership / Subscription Pilot
